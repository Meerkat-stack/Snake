# Irrational Snake

Gra zręcznościowo-edukacyjna będąca wariacją klasycznego "Snake'a". Celem gracza jest zbieranie kolejnych cyfr rozwinięć dziesiętnych znanych stałych matematycznych (Pi, e, pierwiastki itp.).

## Funkcjonalności
* **Edukacja:** Nauka cyfr liczb niewymiernych poprzez zabawę.
* **System kont:** Logowanie, rejestracja i zapisywanie postępów (czas gry, przebyty dystans).
* **8 Trybów gry:** Różne stałe matematyczne do wyboru (π, e, φ, √2, √3, √5 itd.).
* **Statystyki:** Śledzenie najlepszych wyników i przyczyn porażek.

## Technologie
* **Język:** C++
* **Biblioteka:** SFML 3.0.2
* **Środowisko:** Visual Studio Code
